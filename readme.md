# Biorobotic Arm

This is a collaborative project for the course "biorobotics and locomotion".
It contains the source files for the firmware, control-software and remote software.

## General structure
- class Robot
	- Base
    - Shoulder
    - Elbow
    - Wrist
    - Grip
        - open()
        - close()

## API
Close / Open the grip
''''GRIP OPEN / CLOSE''''
